## Overview

Fluent (Dark) Theme for OBS Studio is a theme package that refreshes OBS Studio with a modern Fluent-oriented interface. It targets users who prefer a darker, more refined UI and want a presentation that echoes the cleaner visual language seen in Windows 11 and WinUI 3, while still fitting a cross-platform OBS setup.

The theme is intended for OBS Studio v28 and newer. It includes Qt6 support along with custom SVG icons and supporting assets to help maintain a uniform appearance. It suits streamers and creators who want OBS to feel more polished and contemporary without altering the core application workflow.
